# Catalogue Management & Embedded Quote Request System

## Overview
The Catalogue Management system allows organizations to publish products/services as a public, shareable, embeddable catalog that captures quote requests.

## Core Flow
Products → Catalogue → Quote Request → Inbox → Quotation

## Features
- Product management (name, description, pricing, images, categories)
- Public catalog pages
- Quote request form
- Conversion to quotation

## Strategic Value
Transforms app into a sales intake + quoting platform.
